6//! # Log Engine
7//! Log Engine is responsible for log all authz and init events.
8//!
9//! ## Cedarling log types
10//!
11//!  In [Cedarling-Nativity-Plan Bootstrap properties](https://github.com/JanssenProject/jans/wiki/Cedarling-Nativity-Plan#bootstrap-properties) we have variable `CEDARLING_LOG_TYPE`
12//!  and config type [LogType](`crate::models::log_config::LogType`) that may contain next values:
13//!
14//!  * off
15//!  * memory
16//!  * std_out
17//!  * lock
18//!  
19//!  #### Log type `off`
20//!  
21//!  This log type is do nothing. It means that all logs will be ignored.
22//!  
23//!  #### Log type `memory`
24//!  
25//!  This log type holds all logs in database (in memory) with eviction policy.
26//!  
27//!  #### Log type `std_out`
28//!  
29//!  This log type writes all logs to `stdout`. Without storing or additional handling log messages.
30//!  [Standart streams](https://www.gnu.org/software/libc/manual/html_node/Standard-Streams.html).
31//!  
32//!  #### Log type `lock`
33//!  
34//!  This log type will send logs to the server (corporate feature). Will be discussed later.
35//!
36//!  ## Log Strategy
37//!
38//!  We use [`LogStrategy`] to implement all types of logger under one interface.
39//!
40//!  ## Interfaces
41//!
42//!  Currently we have 2 interfaces (traits):  
43//!
44//!  * `LogWriter` (not public) it is used to write logs.
45//!
46//!  All log implementation should implement this.
47//!
48//!  * [`LogStorage`] is used to gettting logs from log storage.
49//!
50//!  Currently only [MemoryLogger](`memory_logger::MemoryLogger`) implement this.

94/// Initialize logger.
95/// entry point for initialize logger
96pub(crate) async fn init_logger(
97    config: &LogConfig,
98    pdp_id: PdpID,
99    app_name: Option<ApplicationName>,
100    lock_config: Option<&LockServiceConfig>,
101) -> Result<Logger, InitLockServiceError> {
102    let logger = Arc::new(LogStrategy::new(config, pdp_id, app_name)?);
103    let logger_weak = Arc::downgrade(&logger);
104    if let Some(lock_config) = lock_config {
105        let lock_service = LockService::new(pdp_id, lock_config, Some(logger_weak)).await?;
106        logger.set_lock_service(lock_service);
107    }
108    Ok(logger)
109}
